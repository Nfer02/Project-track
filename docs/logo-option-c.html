Construction Notes

Design Rationale

  • P and T share a single vertical stem, merging two letters into one unified mark
  • T crossbar spans the full width at the top, creating a strong horizontal anchor
  • P bowl extends right from the stem just below the crossbar, with a negative-space interior cutout
  • Both letters readable simultaneously: top-down reads T, left-to-right reads P
  • Built on a 64-unit grid for sub-pixel precision at standard icon sizes
  • Optimized small-size variant uses thicker strokes for legibility at 16px
  • Single flat fill, no gradients — works in any monochrome context
