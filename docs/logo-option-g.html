Design Notes

Concept: A smooth upward-trending financial curve flows behind bold "PT" initials, evoking a premium fintech dashboard. The gradient-filled area beneath the curve adds depth without overwhelming the letterforms.

Curve: SVG cubic bezier for an organic, hand-crafted feel. Four data-point nodes sit along the line to reinforce the chart metaphor. The highlighted node (with a glow ring) draws attention to the peak.

Grid: Very faint orthogonal lines (4-6% opacity) simulate a real charting grid, adding texture without visual noise.

Colors: Primary blue (#3b82f6) with lighter accent (#60a5fa). Area fill uses a top-to-bottom gradient from 50% opacity to transparent. Text is near-white (#f1f5f9) for maximum contrast on the dark container.

Scalability: At 32px the grid and smallest nodes are omitted; the curve stroke is thickened so it remains visible. The design holds well from favicon size up to hero banner.
